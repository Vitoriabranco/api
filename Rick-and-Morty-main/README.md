Página Web para consumo de uma API de [Rick and Morty](https://rickandmortyapi.com/).

## Rodando localmente

Clone o projeto

```bash
  git clone https://github.com/ApenasoVini/Rick-and-Morty.git
```

Instale as dependências

```bash
  npm install
```

Abra a página

```bash
  npm start
```
